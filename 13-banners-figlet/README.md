# Banners and Figlet Fonts

Each file contains output samples in one figlet font. Generate your own with the [figlet](http://www.figlet.org/) CLI tool, or online at patorjk.com/software/taag.

| File | Font | Sample words |
|---|---|---|
| [standard.txt](standard.txt) | standard | HELLO, WORLD, WELCOME, ASCII, ART, ERROR, OK, YES, NO, DONE |
| [slant.txt](slant.txt) | slant | HELLO, WORLD, CODE, BUILD, DEPLOY, READY, SHIP IT, WORKING |
| [big.txt](big.txt) | big | HELLO, WELCOME, README, OPEN, CLOSE, START, STOP, WIN |
| [block.txt](block.txt) | block | HELLO, WORLD, BLOCK, ALERT, DEMO, TEST |
| [shadow.txt](shadow.txt) | shadow | HELLO, WORLD, CLOUD, AWS, SHELL, BASH |
| [small.txt](small.txt) | small | HELLO, WORLD, DEBUG, INFO, WARN, FAIL, PASS, 404 |

## Generate your own

```bash
figlet -f slant "your text here"
figlet -f big "your text here"
toilet -f mono12 "your text here"
```

Common figlet fonts: `standard`, `slant`, `big`, `block`, `shadow`, `small`, `script`, `bubble`, `digital`, `mini`, `mono12`, `mono9`, `term`, `wavy`.
