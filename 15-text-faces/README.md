# Text Faces

One-line emoticons and kaomoji. Drop these into chat, code comments, commit messages, or anywhere a face fits on a single line.

| File | Contents |
|---|---|
| [happy.txt](happy.txt) | smiles, grins, laughter, kawaii happy faces |
| [sad.txt](sad.txt) | frowns, crying, distressed |
| [angry.txt](angry.txt) | mad, annoyed, table flips |
| [love.txt](love.txt) | hearts, kisses, hugs, valentines |
| [kaomoji.txt](kaomoji.txt) | shrug, lenny face, disapproval, animals, dance, music, surprised, etc. |
